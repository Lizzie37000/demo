# demo
progress bar demo for singtel test

# dependence
node v0.10.28+

# how to start
npm install
npm start
(excute 'npm install' and 'npm start' in 'package.json' location)

├── public
├── src  
│ ├── api           // request
│ ├── models        // state
│ ├── pages         // component(UI) page
│ ├── index.js      // entrance
│ ├── router.js     // router
├── package.json
├── README.md       // help
